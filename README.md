### Overview

Developed a real-time web chat application focused on low-latency communication between multiple users. The project was designed to demonstrate bidirectional communication using WebSockets, enabling instant message delivery without requiring HTTP polling or page refreshes.

### Technologies

Node.js • WebSockets • HTML5 • CSS3 • JavaScript

### Activities

* Designed and developed a real-time messaging application supporting simultaneous communication between multiple connected users.
* Built the backend using Node.js to manage WebSocket connections, client sessions, message broadcasting, and connection lifecycle events.
* Implemented bidirectional communication through WebSockets, allowing messages to be delivered instantly to all connected clients with minimal latency.
* Developed a lightweight authentication flow based on usernames, enabling users to join chat sessions without requiring account registration.
* Implemented automatic message synchronization across connected clients, ensuring all participants receive updates in real time.
* Developed client-side logic to automatically scroll conversations as new messages arrive, improving the overall user experience.
* Structured the application to support an unlimited number of messages during an active session while maintaining a responsive interface.
* Built responsive user interfaces using HTML5, CSS3, and JavaScript, providing a clean and intuitive messaging experience across different screen sizes.
